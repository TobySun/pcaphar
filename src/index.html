PCAP Web Performance Analyzer

PCAP files contain detailed information about the requests and responses sent over the network. The PCAP Web Performance Analyzer makes it easy to do web performance analysis using a PCAP file.

Using the PCAP Web Performance Analyzer, you can

  • Visualize the HTTP waterfall in a PCAP file using HarViewer
  • Get Page Speed suggestions for a PCAP file
  • Generate a HAR file from a PCAP file

We built the PCAP Web Performance Analyzer to help in analyzing mobile web performance. Please see how to capture mobile traffic for instructions to capture mobile traffic from your mobile device.

Get Started

Upload a PCAP file to get started. You will be able to download a HAR file, view the HTTP waterfall using HarViewer, or get Page Speed suggestions for your network trace.
